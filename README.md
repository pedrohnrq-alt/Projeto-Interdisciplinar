# Sistema-para-gest-o-e-controle-de-doa-o-de-sangue-para-a-rede-de-sa-de-municipal
O sistema conecta rapidamente doadores a pacientes da rede municipal de saúde. Ele identifica tipos sanguíneos compatíveis, facilita a comunicação e agiliza o atendimento em urgências, fortalecendo a solidariedade e contribuindo para salvar vidas.
